    准备矿机

    一般不推荐个人使用的电脑挖矿，特别是笔记本电脑，即使笔记本满足一般的挖矿软件运行要求，但发热量厉害，容易出问题。个人组装的台式机，如果散热不好，也是不推荐的。
    目前多数的挖矿软件对运行的硬件都设定了下限，基本要求是显卡2G内存以上，一般从耗电量和收益等综合来说，推荐配置专业显卡矿机比较划算，(6G以上显存，1500HM以上频率，6卡）
    对于红烧肉来说，目前软件支持比较好是N卡（后续A卡应该也有超级软件出来），这里有个简单对照表（网络数据）可供参考
    显卡型号 单卡算力 6卡算力

    NVIDIA(英伟达)1060 6G ~8m 45m-52m
    NVIDIA(英伟达)1070 8G ~12.05m 72m~80m

    设置挖矿软件

    91pool官方群提供了几个热门挖矿软件，大家可以下载使用，具体是否用方法如下。
    特别注意：有些挖矿软件由于加壳等原因，容易被杀毒软件查杀，建议关闭杀毒软件（qq管家，360等），不推荐个人电脑挖矿，本群软件为广大矿友提供，在此只做参考。
    1.查看自己的是AMD卡还是Nvidia卡
    参考下面操作步骤https://jingyan.baidu.com/article/7e440953cec6182fc1e2ef57.html
    2.解压下面我们提供的挖矿软件包

    A卡和N卡，根据自己的情况，选择

    3.配置，我们以N卡为例

    用【记事本】打开这个文件，修改你的钱包地址（记事本使用：https://jingyan.baidu.com/article/e52e3615a5378840c70c5157.html）

    格式是：
    -o 91地址 -u 钱包地址.机器编号 -p x
    这里，-u是用户，-p是秘密，91暂时不设置密码，直接x就可以
    如果想配置一个故障转移的地址，也就是配置2个矿池地址，那么复制这一部分就可以，如下：
    -o 91地址1 -u 钱包地址.机器编号 -p x -o 91地址2 -u 钱包地址.机器编号 -p x
    如下图

    这里注意：
    1.机器名字，自行定义，比如你有3台机器，你可以定义为：ok001,0k002,ok003，方便你从91网站看到自己每台机器的情况，如果机器名字定义一样，91会将他们合算到一起，算一台机器，当然收益肯定是1+1=2
    2.机器名，只能是数字字母，不能还有其他符号，长度10位以内
    修改好后，保存即可
